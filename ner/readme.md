# Многоцелевая модель для NER + event-CLS

Проект посвещён созданию многоцелевой модели для NER-разметки новостных текстов и выделением тэгов о содержимой информации.

## Поставленные цели

Обучить единую модель, для решения задач:
- Извлечение сущностей из новостных текстов (NER)
- Определяет набор характеристик в тексте (CLS)

После обучения провести квантизацию и проанализировать полученные результаты

## Достигнутые результаты

Были проанализированны исходные данные и принято решения в качестве архитектуры использовать BERT с двумя головами. Также был создан отдельный лосс, коорый учитывает обе задачи и самостоятельно проводит масштабирование лоссов от каждой задачи за счёт обучаемых весов. После обучения, был проведён анализ качества прогноза, а модель квантизована. Результаты на тестовом датасете представлены ниже.

| Модель | f1 NER | f1 CLS | Средняя скорость CPU | Средняя скорость GPU |  
| :---------------------- | :---------------------- | :---------------------- | :---------------------- | :---------------------- |
| Полная модель | 0.6867 | 0.7496 | 0.0453 | 0.0138 |  
| Конвертация до float 16 | 0.6867 | 0.7496 | 0.2028 | 0.0141 |  
| Динамическая квантизация (qint8)| 0.4664 | 0.7217 | 0.0271 | - |
