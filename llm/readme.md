# Обучение собственной LLM

В проекте рассмотрены два первых этапа обучения нейросети (Pretrain и SFT).

## Поставленные цели

Провести два этапа обучения большой языковой модели:
- Pretrain - обучить с нуля Большую языковую модель (примерно на 140 млн параметров) для генерации текста на русском языке
- SFT - обучить более мощную модель (Qwen2.5-0.5B) соблюдать стиль инструкций и давать корректные ответы на русском языке

## Достигнутые результаты

В ходе данного проекта мы достигли следующих результатов:
- Модель созданая с нуля на базе LlamaConfig, научилась генерировть приемлемый текст на русском языке. 
- А модель после SFT уверено работает с инструкциями, может выступать в качестве уиного помошника, но она всё ещё слишкм маленькая, чтобы давать корректные ответы на все вопросы.
